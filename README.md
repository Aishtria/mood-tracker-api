# Mood Tracker AI API
A full-stack backend application that tracks user moods and provides AI-driven insights.

## 🚀 Features
- **Mood Logging**: Store daily entries in a MySQL database.
- **AI Analysis**: Integration with Google Gemini/OpenAI to analyze emotional trends.
- **RESTful API**: Clean Express.js routes for data management.

## 🛠️ Tech Stack
- **Node.js** & **Express**
- **MySQL** (Database)
- **Git** (Version Control)

## ⚙️ Setup
1. Clone the repo.
2. Run `npm install`.
3. Configure your `.env` file.
4. Start the server with `node server.js`.
